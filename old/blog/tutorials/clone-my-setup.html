About

This is my personal repository for my (Arch+Gentoo/some other distro's I randomly hop to) dotfiles.

Here is some information about my setup:

  • Window Manager: dwm
  • Compositor: picom
  • Terminal: st
  • Shell: zsh
  • Text editor: neovim (nvchad)
  • Bar: dwmblocks
  • Notifications: dunst
  • Application launcher: dmenu
  • File Manager: none, I just use my terminal

dmenu

I use my own dmenu configuration. For networking I use networkmanager-dmenu-git(AUR)

Fonts

  • Nerd Fonts

These include all fonts I use for my system.

Preinstall

Make a new using without setting a home yet. Now make the home manually:

sh mkdir -p /home/username Then clone my dotfiles:

sh git clone https://github.com/Ahwxorg/dotfiles ~ chmod +x ~/.lcoal/bin/*

Dependency

The main stuff should already be installed if you use my Arch install script.

We also need yay and git:

sh sudo pacman -S --needed git base-devel git clone https://aur.archlinux.org/yay.git cd yay makepkg -si

wallpaper

I keep my wallpapers in my ~/pix/wall folder.

I set my wallpaper with a script called wallpaper-pywall in my ~/.local/bin/ directory.

dwmblocks

sh cd ~/.local/src git clone https://github.com/Ahwxorg/dwmblocks cd dwmblocks doas make clean install

dunst

Copy my dunst configuration into yours:

sh cp -r ~/Downloads/dotfiles/.config/dunst ~/.config/dunst

Add dunst startup to your ~/.config/bspwm/bspwmrc (see mine):

sh dunst -conf $HOME/.config/dunst/dunstrc &

dmenu

sh cd ~/.local/src git clone https://github.com/Ahwxorg/dmenu cd dmenu doas make clean install

picom

sh yay -S picom

keybindings

For keybindings I mainly use pure dwm, but for some special things I use sxhkd, the config file can be found in my dotfiles at ~/.config/sxhkd/sxhkdrc.

In case you are desperately stuck, open an issue or send me an email.
