So I rename my pictures to {1...1000}.$extention

Open vimv

Run the following commands:

sh which vimv doas nvim !! Replace vi with vim or nvim, depending on your editor.

Go into the folder where you want to bulk rename and enter the following Vim command. First remove any other names of any files if there are any, so that you get a list of file name extentions such as this:

sh .png .jpg .jpg .jpg .png You can do this easily using :%s and/or the V-Block editor, which you can use with Ctrl + V.

Now you can enter this command:

sh :%s/^/\=printf('%-4d', line('.'))
