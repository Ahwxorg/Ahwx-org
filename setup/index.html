Software

As my operating system, I mainly use Arch- and Void GNU/Linux due to the AUR for Arch and speed of XBPS for Void, and since they're both a rolling release and are quite simple for me to use. I am currently using Proxmox VE - which is Debian based - on my server.
My IDE/text editor is (Neo)vim.
As for a browser I use a few different browsers for different usecases. For example for most normie work, I use Firefox. For things where I do actually want a lot of privacy, I use LibreWolf and for testing websites which won't work with the Firefox webkit I use Ungoogled Chromium.
I still use Discord like a normie. Ahwx#9514.
I use i3-gaps as my window manager on the X window system.
I use the ZSH shell with ohmyz.sh.
